## Description

[Gambrin](https://erp-be.onrender.com/) Backend Project built with nest js [Nest](https://github.com/nestjs/nest)

## Installation

```bash
# clone the repo
$ git clone https://github.com/nbrucepro/erp
# installing dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
